# UX Flow Review

## Luồng sau khi sửa

- Vào trang lần đầu: hero giải thích SmartBus + 3 CTA chính.
- Tìm tuyến: search/filter tuyến, card có mã, tên, giờ chạy, tần suất, giá vé, CTA bản đồ/chatbot.
- Tìm bến gần tôi: CTA gọi dashboard/GPS, UI hướng dẫn bật vị trí nếu bị chặn.
- Xem bản đồ: layer control/legend/popup ngắn gọn, fallback khi thiếu Leaflet/tile.
- Du lịch: card có tên, tỉnh, mô tả, tuyến/bến gần, CTA hỏi chatbot.
- Chatbot: quick replies, loading bubble, fallback backend lỗi.
- Đăng nhập/logout: hiển thị trạng thái user, xóa state khi logout.
- Review/community: người chưa đăng nhập xem public, tạo/vote yêu cầu login.

## Lỗi UX còn lại

Cần tiếp tục test thực tế với người dùng để tinh chỉnh microcopy và vị trí control bản đồ.
