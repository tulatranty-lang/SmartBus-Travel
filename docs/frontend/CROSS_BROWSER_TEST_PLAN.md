# Cross-browser Test Plan

## Trình duyệt cần test

- Chrome desktop.
- Edge desktop.
- Firefox desktop.
- Android Chrome.
- iPhone Safari nếu có.

## Màn hình

- 1366x768.
- 1920x1080.
- 390x844.
- 768x1024.

## Checklist chức năng

1. Mở trang chủ, đọc được mục tiêu SmartBus.
2. Đăng nhập/đăng ký/logout.
3. Bản đồ load tile.
4. GPS permission hoạt động hoặc có thông báo từ chối.
5. Tìm tuyến, xem popup tuyến.
6. Xem bến, popup bến.
7. Bật/tắt layer tuyến/bến/xe/du lịch.
8. Lọc địa điểm theo tỉnh/thành.
9. Hỏi chatbot và kiểm tra fallback khi backend tắt.
10. Review/community không vỡ layout.
11. Form validation rõ.
12. Sidebar/menu responsive.

## Cách ghi kết quả

Ghi trình duyệt, màn hình, lỗi, file sửa, cách xác nhận và lỗi còn lại.
